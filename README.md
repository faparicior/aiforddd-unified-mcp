# Unified MCP Server

This is the unified MCP server for DDD for AI Tool Suite.

## Usage

You can run this MCP server directly using `npx`:

```bash
npx github:faparicior/aiforddd-unified-mcp
```
(Replace with your actual repository URL/name if different)

Or locally:

```bash
npm install
npm run build
npm start
```

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Build:
   ```bash
   npm run build
   ```

3. Run:
   ```bash
   npm start
   ```
